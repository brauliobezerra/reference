## Strict keywords

> **<sup>Lexer:<sup>**  
> AS        : `as`  
> BOX       : `box`  
> BREAK     : `break`  
> CONST     : `const`  
> CONTINUE  : `continue`  
> CRATE     : `crate`  
> ELSE      : `else`  
> ENUM      : `enum`  
> EXTERN    : `extern`  
> FALSE     : `false`  
> FN        : `fn`  
> FOR       : `for`  
> IF        : `if`  
> IMPL      : `impl`  
> IN        : `in`  
> LET       : `let`  
> LOOP      : `loop`  
> MATCH     : `match`  
> MOD       : `mod`  
> MOVE      : `move`  
> MUT       : `mut`  
> PUB       : `pub`  
> REF       : `ref`  
> RETURN    : `return`  
> SELFVALUE : `self`

> STRUCT    : `struct`

## Keywords reserved for future use

These keywords aren't used yet, but they are reserved for future use:

> **<sup>Lexer</sup>**  
> ABSTRACT : `abstract`  
> ALIGNOF : `alignof`  
> BECOME : `become`  
> DO : `do`  
> FINAL : `final`  
> MACRO : `macro`  
> OFFSETOF : `offsetof`  
> OVERRIDE : `override`  
> PRIV : `priv`  
> PROC : `proc`  
> PURE : `pure`  
> SIZEOF : `sizeof`  
> TYPEOF : `typeof`  
> UNSIZED : `unsized`  
> VIRTUAL : `virtual`  
> YIELD : `yield`  

## Weak keywords

These keywords have special meaning only in certain contexts:

> **<sup>Lexer</sup>**  
> DEFAULT : `default`  
> STATICLIFETIME : `'static`  
> UNION : `union`  
> CATCH : `catch`
